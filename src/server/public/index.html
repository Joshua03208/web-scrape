Guide — what each scraper does

How it works

  1. The scrapers are built in — press a site's Scrape button to run it, or Run now on the Run tab to run every enabled site.
  2. Prices land in Results; spares mappings land in Spares; product photos download via the Images button.
  3. Edit tweaks a site's settings (prefixes, max pages, etc.). The strategy reference below explains each type.

Prefix search best for price lists

Uses the shop's own search box, once per part-number prefix, walking every results page. It also walks the shop's category menus and chases any cross-referenced part numbers it spots, so it catches products the search engine misses. Most thorough option for collecting prices.

Needs: Base URL (shop homepage), Search URL pattern with {query} and {page} placeholders, and your prefixes.
Example — Central Services: base https://central-servicesuk.co.uk/, pattern index.php?route=product/search&search={query}&page={page}&limit=100, prefixes 112., 113., 119., 120., 133., 150., 992., 995.

Category crawl

Skips the search box entirely and walks the shop's category pages (found via its menus and sitemap), reading every product listing. Use when a shop has no usable search, or as a lighter alternative to prefix search. Still needs prefixes to know what counts as a part number.

Needs: Base URL (shop homepage) and prefixes. No search pattern.

Crawl all links last resort

Follows every link on the site and pairs part numbers with the nearest price on each page. Works anywhere but is slower and less precise — only use it when neither of the above fits. Max pages caps how far it wanders.

Needs: Base URL and prefixes.

Spares map no prices

For linking spare-part codes to the product they belong to. Point it at a category page; it opens every product in that category (and its sub-categories) and records the codes from each product's Spares list against the product name and SKU. Results appear in the Spares tab.

Needs: Base URL = the category page (keep any ?product_tag=... part). Prefixes and search pattern stay empty.
Example — Intatec showers: base https://www.intatec.co.uk/product-category/plumbing-and-heating/showers/?product_tag=inspiration-showers

Eric spares upload a CSV

A separate tool on the Eric tab, not a site here. Upload a CSV of products, it looks each one up on the eric supplier feed and returns a CSV of every spare part (code, description, colour, stock). See that tab for details.

Shared settings

Max pages caps how many pages one run may fetch (politeness + runaway protection) — 200 suits big shops, 80 a single category. Dealer login is only for sites that show prices after signing in: give the login page address and your credentials, and the scraper signs in before crawling. Enabled controls whether Run now includes the site; the per-site Scrape button works regardless of other sites.
